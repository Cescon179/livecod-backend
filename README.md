# livecod-backend
The backend server that powers [livecod](https://livecorona.co.kr).

[livecod](https://livecorona.co.kr)의 백앤드 서버입니다.

## Getting Started
### Installation
* Node.js v10+

```shell
git clone https://github.com/LiveCoronaDetector/livecod-backend.git
cd livecod-backend
npm install
```

### Starting the development server
```sh
npm run dev
```
